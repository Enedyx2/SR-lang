Translation Contribution Guide

This folder contains all language files for Survival Reworked.
You are welcome to edit them to help us translate the addon. Some files may contain grammar errors or typos (maybe even this README file).
You can technically translate on mobile, but we recommend doing it on a computer.
This is the 1.8.0 texts file, changes made on this folder will only apply once 1.8.0 comes out.

DO NOT TRY TO TROLL AND ADDING INSULTS, WE'LL BAN YOU!
if you do typos yourself it's not a big deal don't worry :)

Instructions for editing existing language files:

- Have a GitHub account.
- Fork this repository to your computer (aka clone). It’s easier if you use the GitHub Desktop app on computer.
- Edit the lang file (do not touch the id on the left of the "=").
- Create a pull request. Your changes will be reviewed before being merged.

Instructions for adding a new language:

- Have a GitHub account.
- Fork this repository to your computer (aka clone). It’s easier if you use the GitHub Desktop app on computer.
- Copy the en_US language file and rename it according to the new language (see the list below).
- Add the new language to the languages.json file.
- Edit the lang file (do not touch the id on the left of the "=").
- Create a pull request. Your changes will be reviewed before being merged.
